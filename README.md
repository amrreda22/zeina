# Zeina Events

A platform for showcasing event and celebration products (koshat, cakes, invitations, etc.) with filters, product details, and an admin dashboard.

## Features
- Browse products by category
- Filters (price/governorate/city/subcategory)
- Product details page
- Favorites
- Product admin dashboard

## Tech Stack
- HTML, CSS, JavaScript
- Supabase

## Project Structure
- `pages/categories/` category pages
- `pages/products/` product details and favorites
- `pages/public/` public pages
- `js/services/` data/service layer
- `js/pages/` page logic
- `js/features/` shared features
